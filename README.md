- [【実行環境】Rubyに一から入門してみる](https://zenn.dev/tf63/scraps/6c73e15279baaa)

```
    # ruby環境の初期化
    bundle init
    # Gemfileからパッケージをインストール
    bundle install
    # パッケージの追加
    bundle add <package>
```

- 初期化時に`Gemfile.lock`は空っぽのファイルを使えば良さそう
